การใช้งาน Docker เบื้องต้น

Docker ช่วยจำลองสภาพแวดล้อม (Container) ให้โปรแกรมรันได้ทุกที่เหมือนกัน

  • docker build: สร้าง Image จาก Dockerfile
  • docker run: สั่งรัน Container จาก Image
  • docker ps: ตรวจสอบ Container ที่กำลังทำงาน
  • docker push: อัปโหลด Image ขึ้น Docker Hub
